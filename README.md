# Predicción de Reingresos Hospitalarios con R y Power BI
Proyecto completo de análisis de datos hospitalarios ficticios. Incluye limpieza y preparación de datos, análisis descriptivo, clusterización de pacientes mediante k-means, modelos predictivos de reingreso a 30 días (regresión logística y árbol de decisión) y un dashboard interactivo en Power BI para la interpretación ejecutiva de resultados. El objetivo es identificar perfiles de riesgo y apoyar la toma de decisiones en la gestión asistencial.

#Contenido del proyecto
R/ → Script completo del análisis (EDA, clustering, modelos, métricas)

dashboard/ → Dashboard en Power BI

data/ → Datos ficticios utilizados en el estudio

#Técnicas aplicadas
Limpieza y transformación de datos

Análisis descriptivo y visualización

Clusterización (k-means + PCA)

Modelos predictivos: regresión logística y árbol de decisión

Métricas: AUC, ROC, sensibilidad, especificidad, punto de corte óptimo

Dashboard interactivo para KPIs, clusters y perfiles de riesgo

#Conclusiones principales
La edad y la causa de ingreso son las variables más relevantes para predecir reingresos.

El cluster 2 agrupa a los pacientes con mayor riesgo y mayor complejidad asistencial.

La regresión logística ofrece la mejor sensibilidad para identificar pacientes vulnerables.

El dashboard permite una visión clara y accionable del comportamiento asistencial.
