# An-lisis-de-reingresos-hospitalarios-con-R-clusterizaci-n-y-modelos-predictivos.
Análisis de ingresos hospitalarios con R y Power BI: limpieza de datos, EDA, clusterización, modelos predictivos de reingreso a 30 días (logit y árbol) y dashboard interactivo. Proyecto orientado a identificar perfiles de riesgo y optimizar la gestión asistencial.
